## Within this folder, I have published my project using Formula 1 telemetry data for visualization purposes.

After pulling data from the FastF1 API, I created a interactive visualization where users get to choose a year, event, and location and can visualize any track viewing metrics such as:
* braking zones: analyzing key deceleration points
* speed (km/h): identifying high speed corners vs. technical sectors
* throttle %: mapping power application
* DRS (Drag Reduction System): checking deployment zones
* gear shift changes: watching shifting patterns through complex turns

This project uses python and libraries such as Pandas (data manipulation), Matplotlib (visualization), and NumPy (coordinate processing).
